# Changelog

## 1.0.0 (2026-06-08)

### Features

- Initial release
- PAYG balance monitoring (CNY) with breakdown showing charged and granted balance
- Token Plan monitoring with credit usage progress bars
- Auto-detect account type (PAYG vs Token Plan) via API probe
- Usage token auto-sync via built-in WebView login + JS hook
- WebView2 cache scanning for cookie extraction
- Monthly usage breakdown by type (monthly / plan / compensation)
- Auto-refresh at configurable intervals (1min / 5min / 30min / 1hr)
- System tray with left-click toggle and right-click menu
- Frameless transparent window with blur effect
- Launch on startup support
- Manual usage token paste fallback
- Multi-platform support (Windows NSIS + macOS DMG)
